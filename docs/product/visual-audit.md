# Auditoría visual y mejoras

El HTML de referencia contiene 19 pantallas: splash, onboarding, acceso adulto, perfiles, home, mapa, módulos de Matemáticas/Arte/Letras/Lógica, cuento, recompensa, subida de nivel, progreso, perfil, panel adulto y logros. Define morado como identidad, amarillo/naranja para recompensa y colores por área.

Esta implementación conserva esa identidad y aplica cambios de producto:

- Tipografía mínima mayor y layout adaptable a móvil, tablet y escritorio.
- Botones de 56 px o más y foco de teclado visible.
- Menos elementos simultáneos; una consigna y una acción por reto.
- Home con jerarquía clara: continuar, elegir aventura, reto diario.
- Error tratado como pista y reintento, no como castigo.
- Acceso adulto protegido con una operación matemática.
- Preferencia de movimiento reducido y soporte del sistema operativo.
- Sin chat, anuncios, compras, cámara, micrófono ni ubicación.
