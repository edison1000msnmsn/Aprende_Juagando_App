CODIGO FUENTE DEL PROYECTO
==========================

El codigo fuente completo NO se duplica dentro de esta carpeta para evitar
copias repetidas y confusas.

El codigo fuente completo esta en la raiz del repositorio descargado desde GitHub.

Carpetas principales:

- frontend/mobile/       Aplicacion movil Flutter para Android.
- backend/               API REST desarrollada con NestJS.
- database/prisma/       Modelo de base de datos, migraciones y seed.
- content/               Niveles y actividades educativas por mundo.
- docs/                  Documentacion del proyecto.
- packages/              Validador de contenido.

Archivos importantes:

- package.json
- docker-compose.yml
- README.md
- railway.json
