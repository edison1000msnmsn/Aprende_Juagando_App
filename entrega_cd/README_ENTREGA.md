# Entrega CD - AprendeJugando

Esta carpeta contiene los archivos directos para la entrega académica:

- `01_APK/AprendeJugando.apk`: instalador Android.
- `02_CODIGO_FUENTE/README_CODIGO_FUENTE.txt`: explicación de ubicación del código fuente.
- `03_DOCUMENTACION/guia-interactiva-arquitectura.html`: guía/presentación interactiva del proyecto.
- `04_INSTRUCCIONES/COMO_EJECUTAR.txt`: pasos para instalar, ejecutar y revisar.

El código fuente completo está en la raíz del repositorio descargado.
