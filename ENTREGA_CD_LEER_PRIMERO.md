# Entrega en CD - AprendeJugando

Este repositorio contiene el aplicativo completo **AprendeJugando** para revisión académica.

## Descarga sin cuenta de GitHub

El profesor puede descargar todo sin iniciar sesión:

1. Entrar al repositorio público:
   <https://github.com/edison1000msnmsn/Aprende_Juagando_App>
2. Presionar **Code**.
3. Presionar **Download ZIP**.
4. Descomprimir el ZIP.
5. Copiar la carpeta descomprimida completa al CD.

## Qué entregar en el CD

La carpeta importante para la entrega es:

```text
entrega_cd/
```

Contiene:

```text
entrega_cd/
├── 01_APK/
│   └── AprendeJugando.apk
├── 02_CODIGO_FUENTE/
│   └── README_CODIGO_FUENTE.txt
├── 03_DOCUMENTACION/
│   ├── guia-interactiva-arquitectura.html
│   └── README_PROYECTO.md
└── 04_INSTRUCCIONES/
    └── COMO_EJECUTAR.txt
```

El **código fuente completo** está en la raíz del repositorio:

```text
backend/
database/
content/
frontend/mobile/
docs/
packages/
docker-compose.yml
package.json
README.md
```

## APK final

El APK para instalar en Android está en:

```text
entrega_cd/01_APK/AprendeJugando.apk
```
